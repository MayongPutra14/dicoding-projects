## 🤖 AI Usage in This Project

Dalam proyek ini, saya menggunakan beberapa tools AI untuk membantu proses pengembangan secara lebih efisien dan terarah:

* **GPT (ChatGPT)**
  Digunakan untuk:

  * Menjelaskan langkah-langkah pengerjaan proyek
  * Menerjemahkan dan memahami kriteria dari tim Dicoding
  * Membantu memahami konsep yang belum jelas


* **DeepSeek**
  Digunakan untuk:

  * Membantu menyesuaikan kode dari modul *Belajar Fundamental Aplikasi Web dengan React*
  * Menerapkan konsep dari modul ke dalam proyek pribadi
  * Mengecek kualitas kode
  * Memberikan saran refactoring
  * Merekomendasikan perbaikan logika dan fitur

Dengan kombinasi tools ini, saya bisa lebih fokus memahami konsep, memperbaiki kode, dan mengembangkan fitur dengan lebih terarah.
